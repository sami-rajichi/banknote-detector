Comprehensive Analysis of YOLO Model Performance

üìã Executive Summary

This report presents the results of four comprehensive experiments conducted to develop an optimal banknote detection system using state-of-the-art YOLO models. The evaluation focused on accuracy, precision, recall, and practical deployment considerations.

Key Finding: YOLO11x demonstrated the best balance of accuracy and practical performance, achieving 98.37% mAP50 while maintaining robust detection capabilities with minimal false positives.

‚öôÔ∏è Standardized Inference Configuration

All models were evaluated using consistent inference parameters to ensure fair comparison:

Confidence Threshold: 0.6     IoU Threshold: 0.55     Device: CUDA     Precision: AMP + Half Precision

üî¨ Methodology & Approach

Objective: Develop a robust banknote detection system capable of identifying banknotes in various scenarios (near/far, different lighting conditions, multiple orientations).

Experimental Design:

  • Dataset: Custom annotated banknote dataset with diverse scenarios
  • Model Architecture: Comparison between YOLO11 (l/x) and YOLO12l variants
  • Training Strategy: Transfer learning with backbone freezing for efficient training
  • Evaluation Criteria: mAP50, mAP50-95, Precision, Recall, and qualitative performance

Key Innovations:

  • Optimized augmentation strategies for small object detection
  • Multi-stage fine-tuning approach
  • Single-class optimization for banknote-specific detection
